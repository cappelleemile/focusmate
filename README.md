# FocusMate – Your Minimal AI Daily Planner 🧠✨

FocusMate is a lightweight, distraction-free web app that helps you plan your day, prioritize tasks, and stay focused — powered by AI.

🚀 Features:
- Simple input: "What do you want to achieve today?"
- AI-generated daily schedule based on your goals
- Task prioritization with smart suggestions
- Clean Pomodoro timer for focused work sessions
- Daily reflection prompt to keep you intentional

⚙️ Built with:
- HTML, Tailwind CSS, JavaScript
- OpenAI API (GPT) for AI logic
- No frameworks. No login. Just focus.

📱 Mobile-friendly • Fast • Open Source
